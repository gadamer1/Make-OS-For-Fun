# Make-OS-For-Fun
make os for fun 

---

### first, install those packages
```
apt-get install nasm   // for compile asm
apt-get install vim    // ide
apt-get install qemu   // virtual emulator for run os 
```

### second, run makefile operands
```
make
make exec
```
